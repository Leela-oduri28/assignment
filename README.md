# Frontend Assignment

A React + Vite application implementing product listing, search with debounce, pagination, and product management (add/edit) using in-memory state.  
This project fulfills all the requirements provided in the assignment.

## Features Implemented

### 1. Product List Display
- Table View (List format)
- Card View (Grid layout)
- Toggle between Table â†” Card view
- Clean UI with TailwindCSS

### 2. Search with 500ms Debounce
- Real-time search across product names
- Debounced at 500ms
- Works in both table & card views

### 3. Add & Edit Product
- Product form with fields:
  - Name (required)
  - Price (required, number)
  - Category (required)
  - Stock (optional)
  - Description (optional)
- Inline validation with red error messages
- Edit mode autofills the form
- Updates stored in memory (no backend)

### 4. Pagination
- Next / Previous buttons
- Pagination for both views
- Stable table layout (fixed widths)
